# <a name="sharepoint-site-branding-and-page-customization-solutions"></a>Сайт фирменной символики и страница центра развертывания решений SharePoint

Используйте модель страницы SharePoint и вариантов оформления, модуль SharePoint 2013 темы и CSS для фирменной символики сайта SharePoint и страницы.

_**Область применения:** Office 365 | SharePoint 2013 | SharePoint Online_

Можно настроить внешний вид и функции сайта SharePoint двумя способами:

- С помощью модуля темы для создания пользовательских тем (вариантов оформления в SharePoint 2013 и SharePoint Online). Как минимум тем определения цветов. Полный тема определяет цвета, шрифты, фонового изображения и связанных главной страницы и .preview-файл, определяющий, как просмотреть главную страницу. Удаленный подготовки шаблон можно использовать для применения темы к сайтам.

- Необходимо создать пользовательские каскадные таблицы стилей (CSS) для применения к сайтам SharePoint Online. Приложения для SharePoint и удаленных подготовки шаблон можно использовать для подготовки сайтов SharePoint с помощью настраиваемого CSS.

Фирменная символика изменения в диапазоне от недорогих и простых дорогого и комплексного. Пользователи могут использовать пользовательский Интерфейс для применения вариантов оформления, включая фонового изображения, цветовой палитры, шрифты, главной страницы и файл связанного .preview для главной страницы. Модуль SharePoint 2013 темы можно использовать для создания вариантов оформления и подготовка сайтов, и можно создавать настраиваемые CSS для изменения внешнего вида сайта и его элементы.

**Важные**  Можно создать настраиваемые главные страницы и другие структурные элементы в рамках пользовательского брендинга проекта, но долгосрочного затрат на поддержку настраиваемых главных страниц и других настраиваемых элементов структурные высокой. Настраиваемый фирменный стиль можно сделать его более высоких для вашей организации применить обновления и обеспечение непрерывной поддержки.

В этом разделе выполняется построение знания о [разработки в SharePoint и средства разработки и рекомендации](SharePoint-development-and-design-tools-and-practices.md), показывающие, как настроить внешний вид и функции сайта SharePoint.
