# <a name="excel-services-error-codes"></a>Коды ошибок для служб Excel

Службы Excel создает ошибок и сообщения об ошибках на основании ошибки, которые возникают в Службы Excel исключений SOAP. В следующей таблице перечислены ошибки, которые доступны при вызовы методов Веб-службы Excel исключения SOAP. 
  
    
    

You use the  [SubCode](http://msdn.microsoft.com/library/frlrfSystemWebServicesProtocolsSoapExceptionClassSubCodeTopic.aspx) property of the **SoapException** class to capture the error codes. For more information about using the **SubCode** property to capture error codes, see [How to: Use the SubCode Property to Capture Error Codes](how-to-use-the-subcode-property-to-capture-error-codes.md) For more information about Службы Excel alerts, see  [Excel Services Alerts](excel-services-alerts.md). 
  
    
    


## <a name="error-codes"></a>Коды ошибок

В следующей таблице перечислены коды ошибок для оповещения Веб-службы Excel и связанные сообщения, описание и способы решения. 
  
    
    


|**Код ошибки**|**Message**|**Объяснение**|**Решение**|
|:-----|:-----|:-----|:-----|
|ApiInvalidArgument  <br/> |Недопустимое значение аргумента: {0}  <br/> |При вызове API было передано недопустимое значение аргумента.  <br/> 0 = имя аргумента. Имеет недопустимое значение.  <br/> |Используйте допустимое значение для аргумента.  <br/> |
|ApiInvalidCoordinate  <br/> |Координата "{0}" аргумента {1} является недопустимой.  <br/> |0 = имя координаты (строка, столбец, высота, ширина).  <br/> 1 = имя аргумента, имеющего структуру координаты.  <br/> Содержимое класса **RangeCoordinates** или параметры ряд\\столбец\\высота\\ширина или являются недопустимыми. <br/> |Используйте допустимые значения координат для аргумента.  <br/> |
|DimensionAndArrayMismatch  <br/> |Размер указанного массива не соответствует размеру и форме конечного диапазона.  <br/> |Вызывающий попытался задать диапазон в книге, но параметр, содержащий массив значений не входит в указанный диапазон.  <br/> |Убедитесь, что размер указанного массива соответствует размерам конечного диапазона (например, 2 столбца в ширину 3 строки высокой).  <br/> |
|DiscontiguousRangeNotSupported  <br/> |Запрос диапазона не ссылается на непрерывный диапазон. Службы Excel поддерживают только непрерывные диапазоны.  <br/> |Звонящий указано несмежных при попытке задание или получение диапазона ячеек. Службы Excel не поддерживает несмежных диапазонов. Поддерживает только непрерывные диапазоны.  <br/> |Введите непрерывном диапазоне ячеек, например, "A1:B7" или "A1" или "MyTable [#Data]", а не диапазон, например, "A1:B7 B12" или "A1, A3".  <br/> |
|ExternalDataRefreshFailed  <br/> |Не удается получить внешние данные для следующих подключений:  <br/> {0}  <br/> Источник данных может быть не доступен или не отвечает, или к нему закрыт доступ.  <br/> |Попытки обновить источник данных внутри книги не удались.  <br/> 0  это список имен подключений, разделенных \\n.  <br/> |Убедитесь, что источник данных доступен и иметь разрешение на доступ к нему.  <br/> |
|FileOpenAccessDenied  <br/> |У вас нет необходимых разрешений для открытия этого файла в службах Excel.  <br/> |Сбой при вызове метода **OpenWorkbook**, так как пользователь не имеет доступа к файлу. <br/> |Обратитесь к администратору.  <br/> |
|FileCorrupt  <br/> |Выбранный файл не может быть открыт, так как он поврежден, защищен службой управления правами на доступ или имеет формат, не поддерживаемый службами Excel. Может быть этот файл можно открыть в Excel.  <br/> |Сбой при вызове метода **OpenWorkbook**, так как файл поврежден. <br/> |Попробуйте снова открыть файл или используйте Excel для открытия файла.  <br/> |
|FileOpenNotFound  <br/> |Выбранный файл не найден. Проверьте правильность написания имени файла и проверьте правильность расположения.  <br/> |Сбой при вызове метода **OpenWorkbook** файл не существует. <br/> |Убедитесь в том, что файл не был переименован, перемещен или удаленное, что файл находится в надежном расположении и иметь доступ к файлу. Если проблема повторяется, обратитесь к администратору.  <br/> |
|FileOpenSecuritySettings  <br/> |В настоящее время из-за параметров безопасности для служб Excel не удается открыть выбранный файл.  <br/> |Сбой при вызове метода **OpenWorkbook** параметров безопасности администратора его открытие запрещено по различным причинам. Например, файл слишком велик, то есть его размер превышает предел, установленный администратором.<br/> |Обратитесь к администратору.  <br/> |
|FormulaEditingNotEnabled  <br/> |Изменение формул недоступно в этой версии служб Excel.  <br/> |Вызывающий попытался ввести формулу в книгу.  <br/> |Не пытайтесь писать формулу, так как он не поддерживается в этой версии служб Excel.  <br/> |
|GenericFileOpenError  <br/> |Ошибка при открытии выбранного файла.  <br/> |Службы Excel не могут открыть файл по неизвестным причинам.  <br/> |Подождите несколько минут и попробуйте снова открыть файл. Если проблема повторяется, обратитесь к администратору.  <br/> |
|InvalidSheetName  <br/> |Запрошенный лист отсутствует в этой книге.  <br/> |Имя листа не найдено или недопустимо.  <br/> |Используйте допустимое значение для имя листа.  <br/> |
|InvalidOrTimedOutSession  <br/> |Выбранная операция не может быть сейчас выполнена, так как сеанс на сервере завершен. Книгу можно загрузить заново и создать новый сеанс, но все внесенные изменения будут потеряны.  <br/> |Значение  _sessionId_ звонок либо недопустимо или время истекло. <br/> |Перезагрузите книгу в новом сеансе.  <br/> |
|IRMedWorkbook  <br/> |Запрошенный книги  IRM-защиту. Службы Excel не удается загрузить книги IRM-защиту.  <br/> |Сбой при вызове метода **OpenWorkbook**, так как книга защищена службой управления правами на доступ к данным (IRM). <br/> |Передайте только в книгах, которые не являются IRM-защиту.  <br/> |
|MaxSessionsPerUserExceeded  <br/> |Превышено максимальное разрешенное число сеансов для одного пользователя. Выполнение операции невозможно.  <br/> |Превышено максимальное число сеансов, которые пользователь может был открыт в любой момент времени. Это ограничение устанавливается администратором.  <br/> |Не должна превышать ограничение или обратитесь к администратору.  <br/> |
|MultipleRequestsOnSession  <br/> |Операция уже выполняется в данном сеансе. В сеансе может одновременно выполняться только одна операция.  <br/> |Было произведено несколько запросов в ходе одного сеанса. В сеансе может одновременно обрабатываться только один запрос (за малым исключением).  <br/> |Попробуйте выполнить операцию еще раз.  <br/> |
|NotMemberOfRole  <br/> |Доступ запрещен. У вас нет разрешений на выполнение этого действия или на доступ к этому ресурсу.  <br/> |Вызывающий не имеет разрешения на доступ к серверу.  <br/> |Обратитесь к администратору.  <br/> |
|ObjectTypeNotSupported  <br/> |Один или несколько указанных типов объектов не поддерживаются службами Excel. Выполнен откат операции.  <br/> |Вызывающий попытался ввести неподдерживаемые объект типа значения в диапазоне.  <br/> |Повторите операцию, используя один из поддерживаемых типов объектов.  <br/> |
|OperationCanceled  <br/> |Выполнение операции было прервано.  <br/> |Операция, которая в настоящее время выполняется отменяется, так как пользователь вызывает метод **CancelRequest**. <br/> |Вызовите метод **CancelRequest** только в том случае, если требуется отменить текущей операции. <br/> |
|RangeParseError  <br/> |Службам Excel не удалось разобрать запрос диапазона.  <br/> |Не удалось разобрать диапазон, переданный в метод с суффиксом A1 ( **SetCellA1**, **SetRangeA1**, **GetCellA1**и **GetRangeA1**).  <br/> |Введите ссылку на диапазон в формате A1, например "Sheet1! Range("a6:a15") "или допустимую структурированную ссылку, например," [ГородПолучателя]. [#Headers] ".  <br/> |
|RangeRequestAreaExceeded  <br/> |Площадь запрошенного диапазона превышает 1 000 000 ячеек.  <br/> |Запрошенный диапазон превышает ограничение в 1 000 000 ячеек.  <br/> |Для возврата диапазонов, которые содержат более 1 000 000 ячеек, используйте несколько вызовов.  <br/> |
|RetryError  <br/> |Службам Excel не удается обработать запрос.  <br/> |Иногда службы Excel имеют недостаток ресурсов. Когда подобное происходит, службы могут перестать принимать запросы.  <br/> |Подождите несколько минут и повторите для выполнения этой операции еще раз.  <br/> |
|SaveFailed  <br/> |Произошла ошибка при сохранении файла.  <br/> |Сбой при вызове метода **GetWorkbook**. <br/> |Попробуйте сохранить файл еще раз.  <br/> |
|SetRangeFailure  <br/> |Запрошенная операция пыталась перезаписать содержимое ячеек, изменение которых невозможно.  <br/> |Вызывающий попытался записи значений в диапазоне, который защищен ячеек. К примеру ячейка с формулой.  <br/> |Только пустые ячейки или ячейки, содержащие значения можно редактировать службами Excel.  <br/> |
|SheetRangeMismatch  <br/> |Лист, указанный в аргументе "лист", не совпадает с листом, указанным в аргументе "диапазон".  <br/> |Имя листа, переданное для параметра  _sheetName_ не соответствует расположение таблиц, указанный в параметре _rangeName_. <br/> |При указании листа в диапазоне и листа аргументов, убедитесь, что имена листов совпадают с. Например,  `Calculate(Sheet1, Sheet1!Range("A1"))`.<br/> |
|SpecifiedRangeNotFound  <br/> |Запрошенный диапазон не определен на этом листе.  <br/> |Не удалось найти диапазон, переданный в метод с суффиксом A1 ( **SetCellA1**, **SetRangeA1**, **GetCellA1**и **GetRangeA1**).  <br/> |Убедитесь в том, что существует диапазона, указанного в таблице.  <br/> |
|WorkbookNotSupported  <br/> |Не удается открыть выбранный файл, так как он содержит компоненты, которые не поддерживаются службами Excel. Один или несколько следующих неподдерживаемых возможностей были обнаружены в книге:  <br/> {0}  <br/> |Книга содержит не поддерживаемые функции.  <br/> 0 = список имен не поддерживаемых функций, разделенных \\n.  <br/> |Убедитесь в том, что книга не содержит функции, которые не поддерживаются службами Excel.  <br/> |
